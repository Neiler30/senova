Guía de Uso del Sistema SENOVA

Bienvenido al manual de usuario. Aquí encontrarás toda la información necesaria para navegar y utilizar las herramientas de la plataforma de manera efectiva.

Introducción al Sistema SENOVA

El sistema SENOVA (Sistema Escolar de Nueva Organización y Valoración Académica) es una plataforma de gestión académica y administrativa desarrollada en PHP. Está diseñada para facilitar el manejo de usuarios, matrículas, estudiantes, asistencia y reportes dentro de un entorno escolar.

El acceso al sistema está controlado por roles, lo que significa que las opciones, módulos y acciones que verás dependerán de si eres Administrador, Secretaria o Docente.

Acceso al Sistema y Primeros Pasos

¿Cómo obtengo mis credenciales?

Para ingresar al sistema SENOVA debes ser registrado por un administrador. Durante la creación de tu cuenta se asignan tus credenciales (usuario y contraseña) y tu rol (Administrador, Secretaria o Docente).

Una vez recibas las credenciales, dirígete al inicio de sesión e ingrésalas para acceder a tu panel.

Acceso de administrador (demo)
  • Usuario: admin (se completa como admin@senova.org)
  • Contraseña: admin123
En el login ingresa solo el nombre de usuario; el sistema añade automáticamente @senova.org. en la sesion de Demostración puedes ver la guia de como hacerlo
Requisitos y datos que te solicitarán
  • Para crear el usuario, el administrador necesita tu nombre y apellido, fecha de nacimiento, genero, numero de telefono y rol a asignar (Administrador, Docente o Secretaria).
  • Nombre de usuario sugerido (sin dominio). El correo institucional se compone con dominio fijo @senova.org.
  • la contraseña se puede cambiar en el area de actualizar datos que se encuentra en la barra lateral del dashboard.
Política y recuperación de contraseña
  • Recomendado: mínimo 8 caracteres, combinando letras y números.
  • Si olvidas tu clave, solicita restablecimiento al Administrador del sistema.
  • En el primer inicio, actualiza tu contraseña en el área de "Actualizar Datos" y tambien puedes verifica tus datos de perfil.
Cómo cambiar tu contraseña en "Actualizar Datos"
Guía corta: Ingresa a tu panel, abre "Actualizar Datos", ubica el campo de contraseña, ingresa la nueva y guarda los cambios.

Si tienes inconvenientes para ingresar, valida que estés usando solo el usuario sin dominio; el sistema añade automáticamente @senova.org.

Demostración de acceso

Forma incorrectaEscribir usuario con dominio

Si ingresas el usuario junto con el dominio institucional (por ejemplo, admin@senova.org), el inicio de sesión fallará.

  • No agregues @senova.org manualmente ya el login lo añade automáticamente.
  • Evita copiar y pegar correos completos desde documentos.
Forma correctaEscribir solo el usuario

Escribe únicamente el nombre de usuario (por ejemplo, admin). El sistema completa automáticamente el dominio @senova.org.

  • Respeta mayúsculas/minúsculas si tu usuario las incluye.
  • Si no recuerdas tu usuario, solicítalo al Administrador.

Después de iniciar sesión: ¿Qué puedes hacer?

Según tu rol, tendrás acceso a diferentes módulos y acciones dentro de la plataforma.

Administrador
Acceso total
  • Gestionar usuarios: crear, editar, eliminar y asignar roles.
  • Configurar datos generales del colegio.
  • Organizar el ciclo académico: cursos y asignaturas.
  • Acceder a Gestión escolar: estudiantes, matrículas, asistencia y reportes.
Docente
Académico
  • Actualizar sus datos personales y contraseña.
  • Gestionar el grupo asignado en el módulo Gestión escolar.
  • Tomar asistencia diaria.
  • Registrar y consultar matrículas de su grupo.
  • Generar boletines y reportes de notas.
Secretaria
Administrativo
  • Actualizar sus datos personales y contraseña.
  • Registrar y listar estudiantes.
  • Crear cursos y asignaturas.
  • Consultar reportes (si el administrador lo permite).
  • Sin acceso a crear/eliminar usuarios ni al módulo completo de Gestión.

Acciones Disponibles en el Dashboard

Roles y Permisos — Dashboard Principal

Detalle Rápido

El Dashboard Principal es la primera pantalla que ves al iniciar sesión. Actúa como tu centro de control, ofreciendo acceso rápido a las funciones más importantes según tu rol.

Desde aquí puedes navegar a módulos clave como la gestión de usuarios (si eres admin), la configuración del colegio o la actualización de tus datos personales. Cada rol tiene una vista personalizada que prioriza las herramientas que más necesita.

Administrador

Como Administrador, en el dashboard principal tienes acceso total al sistema. Tu función es supervisar y gestionar todos los aspectos de la plataforma.

Docente

Como Docente, en el dashboard principal puedes actualizar tus datos personales.

Secretaria

Como Secretaria, en el dashboard principal puedes actualizar tus datos personales.

Roles y Permisos — Login y Dashboard del Colegio

Detalle Rápido

El Módulo Colegio es el centro neurálgico para la gestión académica y administrativa específica de la institución. Para acceder, es necesario iniciar sesión con credenciales, en este caso solo la contraseña, el panel le mostrará el nombre del usuario que está intentando entrar.

Una vez dentro, cada rol tiene acceso a herramientas especializadas: los administradores configuran el ciclo académico, las secretarias gestionan estudiantes y los docentes administran sus grupos, calificaciones y asistencia.

El dashboard incluye un widget de bienvenida que exhibe la fecha actual y las condiciones meteorológicas. Además, incorpora un módulo de notas para la creación de recordatorios, y un calendario interactivo que permite la asignación de recordatorios a fechas específicas mediante selección por clic.

Administrador

Como Administrador, en el módulo Colegio tienes acceso total a: Gestión Estudiante, Estudiantes Secretaria, Asignaturas, Cursos, y la opción de volver al dashboard principal.

Docente

Como Docente, en el módulo Colegio solo tienes acceso a Gestión Estudiante.

Incluye: matrículas, asistencias, reportes (boletines).

Secretaria

Como Secretaria, en el módulo Colegio tienes las opciones de Estudiantes Secretaria y crear cursos/asignaturas.

Gestionar usuarios

Desde el panel de Administración puedes crear, editar, eliminar usuarios y asignar roles. Esta función es exclusiva para administradores y permite gestionar todo el personal del sistema.

Pasos para gestionar usuarios:
  1. Accede al módulo lista de usuarios" desde tu dashboard.
  2. Para crear un nuevo usuario, haz clic en "Registrar Usuario".
  3. Rellena los datos personales: nombre, apellido, fecha de nacimiento, género, teléfono.
  4. Asigna un rol: Administrador, Docente o Secretaria.
  5. El sistema genera automáticamente el correo institucional @senova.org.
  6. Guarda los cambios. La contraseña se encripta automáticamente con password_hash().

Para editar o eliminar usuarios existentes, ve a "Lista de Usuarios" y usa los iconos de edición (lápiz) o eliminación (basura). Recuerda que eliminar un usuario es irreversible.

Configurar datos del colegio

Desde el módulo de Configuración del Colegio puedes actualizar toda la información institucional. Esta función es exclusiva para administradores.

Campos que puedes configurar:
  • Nombre del colegio: Nombre oficial de la institución.
  • NIT: Número de identificación tributaria.
  • Teléfono / Celular: Contactos telefónicos.
  • Dirección: Ubicación física del colegio.
  • Correo Electrónico Institucional: Email oficial.
  • Correo Educativo: Email para asuntos académicos.
  • Resolución: Número de resolución educativa.
  • Convenio: Ej. Universidad del Caribe.
  • Código Dane: Identificador DANE.
  • Descripción o Lema: Descripción del colegio.
  • Logo del colegio: Opción de "Cambiar Logo".

Accede desde "Config. colegio" en tu dashboard y guarda los cambios para que se apliquen globalmente.

Módulo: Registrar Usuario
Solo Administradores

Este módulo te permite crear nuevas cuentas en el sistema de manera segura y eficiente.

Pasos para registrar un usuario:

  1. Accede a la opción "Crear Usuario" en tu panel de administración.
  2. Completa los datos personales requeridos (nombre, apellido, usuario, etc.).
  3. Asigna el Rol correspondiente (Administrador, Docente o Secretaria).
  4. El sistema generará automáticamente el correo institucional con dominio @senova.org.
  5. Opcionalmente, sube una imagen de perfil para personalizar la cuenta.
  6. Guarda los cambios. La contraseña se encripta automáticamente para mayor seguridad.
Módulo: Listado de Usuarios
Solo Administradores

Gestiona de forma centralizada todos los usuarios registrados en la plataforma.

Funcionalidades disponibles:

  • Visualiza una tabla completa con nombre, correo y rol de cada usuario.
  • Utiliza el icono para editar la información de un usuario existente.
  • Emplea el icono para eliminar un usuario, con confirmación de seguridad previa.
Módulo: Matrículas
Administradores Secretarias

Módulo esencial para el proceso de matrícula de estudiantes, diseñado para ser intuitivo y seguro.

Proceso de matrícula:

  1. Selección del estudiante: Elige de la lista desplegable; el sistema carga automáticamente los datos personales.
  2. Tipo de matrícula:
    • Nuevo/Interno: Proceso estándar sin pasos adicionales.
    • Traslado: Marca la opción "Traslado" para activar campos adicionales.
  3. Documentos requeridos: Adjunta archivos (PDF, JPG, PNG) para validar el traslado.
  4. Confirmación: Revisa y registra; el sistema confirma el éxito de la operación.
