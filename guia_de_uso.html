Guía de Uso del Sistema SENOVA

Bienvenido al manual de usuario. Aquí encontrarás toda la información necesaria para navegar y utilizar las herramientas de la plataforma de manera efectiva.

Introducción al Sistema SENOVA

El sistema SENOVA (Sistema Escolar de Nueva Organización y Valoración Académica) es una plataforma de gestión académica y administrativa desarrollada en PHP. Está diseñada para facilitar el manejo de usuarios, matrículas, estudiantes, asistencia y reportes dentro de un entorno escolar.

El acceso al sistema está controlado por roles, lo que significa que las opciones, módulos y acciones que verás dependerán de si eres Administrador, Secretaria o Docente.

Acceso al Sistema y Primeros Pasos

¿Cómo obtengo mis credenciales?

Para ingresar al sistema SENOVA debes ser registrado por un administrador. Durante la creación de tu cuenta se asignan tus credenciales (usuario y contraseña) y tu rol (Administrador, Secretaria o Docente).

Una vez recibas las credenciales, dirígete al inicio de sesión e ingrésalas para acceder a tu panel.

Requisitos y datos que te solicitarán
  • Para crear el usuario, el administrador necesita tu nombre y apellido, fecha de nacimiento, genero, numero de telefono y rol a asignar (Administrador, Docente o Secretaria).
  • Nombre de usuario sugerido (sin dominio). El correo institucional se compone con dominio fijo @senova.org.
  • la contraseña se puede cambiar en el area de actualizar datos que se encuentra en la barra lateral del dashboard.
Política y recuperación de contraseña
  • Recomendado: mínimo 8 caracteres, combinando letras y números.
  • Si olvidas tu clave, solicita restablecimiento al Administrador del sistema.
  • En el primer inicio, actualiza tu contraseña y verifica tus datos de perfil.

Si tienes inconvenientes para ingresar, valida que estés usando solo el usuario sin dominio; el sistema añade automáticamente @senova.com.

Acceso de administrador (demo)
  • Usuario: admin (se completa como admin@senova.com)
  • Contraseña: admin123
En el login ingresa solo el nombre de usuario; el sistema añade automáticamente @senova.com.

Demostración de acceso

Forma incorrectaEscribir usuario con dominio

Si ingresas el usuario junto con el dominio institucional (por ejemplo, admin@senova.com), el inicio de sesión fallará.

  • No agregues @senova.com manualmente.
  • Evita copiar y pegar correos completos desde documentos.
Forma correctaEscribir solo el usuario

Escribe únicamente el nombre de usuario (por ejemplo, admin). El sistema completa automáticamente el dominio @senova.com.

  • Respeta mayúsculas/minúsculas si tu usuario las incluye.
  • Si no recuerdas tu usuario, solicítalo al Administrador.
  • Tras ingresar por primera vez, actualiza tu contraseña.

Después de iniciar sesión: ¿Qué puedes hacer?

Según tu rol, tendrás acceso a diferentes módulos y acciones dentro de la plataforma.

Administrador
  • Gestionar usuarios: crear, editar, eliminar y asignar roles.
  • Configurar datos generales del colegio.
  • Organizar el ciclo académico: cursos y asignaturas.
  • Acceder a Gestión: matrículas, asistencia y reportes.
Docente
  • Gestionar el grupo asignado en el módulo Gestión.
  • Tomar asistencia diaria.
  • Registrar y consultar matrículas de su grupo.
  • Generar boletines y reportes de notas.
Secretaria
  • Registrar y listar estudiantes.
  • Crear cursos y asignaturas.
  • Consultar reportes (si el administrador lo permite).
  • Sin acceso a crear/eliminar usuarios ni al módulo completo de Gestión.

Roles y Permisos

Como Administrador, tienes acceso total al sistema. Tu función es supervisar y gestionar todos los aspectos de la plataforma.

Acciones Principales:
  • Crear, editar y eliminar usuarios de cualquier rol.
  • Configurar los datos generales del colegio.
  • Gestionar el ciclo académico completo: crear cursos, asignar asignaturas y registrar estudiantes.
  • Controlar matrículas, tomar asistencia y generar reportes.
  • Acceso completo al Módulo Colegio y todas sus funcionalidades.

Como Secretaria, tu rol es administrativo y de apoyo a la gestión académica. Tienes permisos para gestionar la información de los estudiantes y la estructura académica.

Acciones Principales:
  • Registrar y listar estudiantes en el sistema.
  • Crear y organizar los cursos y asignaturas del colegio.
  • Consultar reportes (si el administrador te ha dado permiso).
Restricciones:
  • No puedes crear o eliminar usuarios (como docentes o administradores).
  • No tienes acceso al módulo de "Gestión", que incluye matrículas y asistencia.

Como Docente, tu rol es académico y estás a cargo de tu grupo de estudiantes. Tienes acceso a las herramientas necesarias para gestionar tus clases y el progreso de tus alumnos.

Acciones Principales:
  • Acceder al módulo de "Gestión" para manejar tus grupos.
  • Tomar asistencia diaria de tus estudiantes.
  • Registrar y consultar las matrículas de tu grupo asignado.
  • Generar boletines y reportes de notas de tus alumnos.
Restricciones:
  • No puedes crear, editar o eliminar perfiles de estudiantes.
  • No tienes acceso a la configuración general del colegio ni a la creación de usuarios.

Guía de Módulos Principales

Módulo: Registrar Usuario

Solo Administradores

Este módulo te permite crear nuevas cuentas en el sistema. Para registrar a alguien, sigue estos pasos:

  1. Ve a la opción "Crear Usuario" en tu panel.
  2. Rellena los datos personales (nombre, apellido, usuario, etc.).
  3. Asigna un Rol (Admin, Docente o Secretaria). El sistema generará automáticamente un correo @senova.com.
  4. Opcionalmente, puedes subir una imagen de perfil.
  5. Haz clic en guardar. El sistema usará password_hash() para proteger la contraseña y te mostrará una confirmación con los datos creados.

Módulo: Listado de Usuarios

Solo Administradores

Aquí puedes ver, editar o eliminar a todos los usuarios registrados en la plataforma.

  • La tabla muestra el nombre, correo y rol de cada usuario.
  • Usa el icono para editar la información de un usuario.
  • Usa el icono para eliminar un usuario. Por seguridad, el sistema te pedirá una confirmación antes de borrarlo.

Módulo: Matrículas

Administradores Secretarias

Este módulo es clave para formalizar el ingreso de los estudiantes. El proceso está diseñado para ser eficiente y seguro.

¿Cómo matricular a un estudiante?
  1. Selecciona el estudiante: Elige un estudiante de la lista desplegable. El sistema cargará automáticamente todos sus datos personales (nombre, documento, edad, etc.) para evitar que los escribas de nuevo.
  2. Identifica el tipo de matrícula:
    • Estudiante nuevo/interno: No se requieren pasos adicionales.
    • Traslado desde otro colegio: Marca la casilla "Traslado". Se activarán campos para que ingreses el nombre del colegio anterior y adjuntes documentos como el boletín académico.
  3. Adjunta documentos (si es traslado): Sube los archivos requeridos (PDF, JPG, PNG). Estos son fundamentales para validar el nivel académico del estudiante.
  4. Confirma la matrícula: Una vez que todos los datos obligatorios estén completos, haz clic en registrar. El sistema guardará la información y te mostrará un mensaje de éxito.
