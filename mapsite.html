Arquitectura y Tecnologías

Arquitectura del Sistema

El sistema sigue una arquitectura tipo MVC simplificada (Modelo-Vista-Controlador):

  • Modelo: Archivos PHP que gestionan consultas y conexión a la base de datos (p. ej., en la carpeta database/).
  • Vista: Archivos HTML/PHP que componen la interfaz de usuario (p. ej., en perfil/, Matricula/).
  • Controlador: Lógica PHP que coordina acciones entre el modelo y la vista, a menudo integrada en los mismos archivos de vista o en scripts dedicados.

Seguridad

El acceso a cada módulo está protegido por la función verificarPermiso(), que valida el rol del usuario contra una lista de roles permitidos, previniendo el acceso no autorizado a rutas restringidas.
